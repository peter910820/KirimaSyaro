# syaroBot

## 四天連假不知道要幹嘛，就隨便亂做的Discord播歌機器人



目前擁有模組:

```
youtube_music_V1: 專門用來撥放YouTube的音樂的模組的第一版
```

指令如下:

```
!join : 讓機器人到發話者的語音頻道\n
!leave : 讓機器人退出語音頻道\n
!play YouTube影片網址 : 增加歌曲到撥放清單並撥放\n
!playlist : 查詢當前撥放清單\n
!pause : 暫停音樂\n
!resume : 恢復音樂
```

暫時不支援撥放清單，然後同個網址絕對不要添加兩次，絕對會出事。打完指令後等他一秒在打下一個指令，不然我也不知道會怎樣(

目前只是測試版
